The World Wide Web

It is written in hyperext mark up language, or HTML.

The major pieces of the web are:
-the browser
-the internet
-servers

Whenever someones uses a browser, like Internet Explorer, the browser sends a HTTP requestto a server via the internet. The server replies with files that the browser displays.


HTML

HTML is made up of:
-text content- what your see
-markup-what it looks like
-references to other documents-such as images, videos, etc.
-links to other pages

Documents are made up of HTML elements. Tags are used to help the computer make the distinction between the different types of tags. This video alks about some different types of tags.


Inline and Block Elements

HTML elements can either be inline or block elements. The difference is that block elements create a sort of invisible box around their contents
